Detailed Description

template<typename real = double>
class SolverVEM3D< real >

Specilized class to solve VEM in 3D.

Complex way of computing B. The boundary term requires to compute an interpolating polynomial on the face. This requires to solve a VEM-like problem on the face to compute the polynomial. It's the computational most expensive part.
