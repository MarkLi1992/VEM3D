Detailed Description

template<long embedded, typename real = double>
class Polygon< embedded, real >

Class to represent a Polygon in a Mesh, embedded in 2D or 3D.

It can be an element of the Mesh in 2D, or a face of a Polyhedron in 3D. The only way to initialize it is to call make_shared_Polygon Area, normal and centroid are computed during initialization and saved. Inheritance from enable_shared_from_this to obtain a shared_ptr from this.

template<long embedded, typename real >
Point< embedded, real > Polygon< embedded, real >::computeCentroid ( )

Compute centroid of a Polygon.

Source: https://en.wikipedia.org/wiki/Centroid#Centroid_of_polygon

For the 3D case the Polygon is projected on one of the xy, xz or yz plane to determine 2 of the components. Then it's projected on an another plane to determine the last component.

template<long embedded, typename real >
Vector< 3, real > Polygon< embedded, real >::computeNormal ( )

Normal computed with Newell's algorithm.

Source: http://www.gamedev.net/topic/416131-calculating-a-polygon-normal/?p=3771628

Nx = (Yn-Yn+1)*(Zn+Zn+1) Ny = (Zn-Zn+1)*(Xn+Xn+1) Nz = (Xn-Xn+1)*(Yn+Yn+1)

In the 2D case it's sufficient to compute the Z component.
