Detailed Description

template<long embedded, typename real = double>
class Polygon< embedded, real >

Class to represent a Polygon in a Mesh, embedded in 2D or 3D.

It can be an element of the Mesh in 2D, or a face of a Polyhedron in 3D. The only way to initialize it is to call make_shared_Polygon Area, normal and centroid are computed during initialization and saved. Inheritance from enable_shared_from_this to obtain a shared_ptr from this.
