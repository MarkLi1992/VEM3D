Detailed Description

template<long embedded, typename real = double>
class Polyhedron< embedded, real >

Class to represent a Polyhedron in 3D.

The only way to initialize it is to call make_shared_Polygon Volume, faces orientation, normal orientation and centroid are computed during initialization and saved. Inheritance from enable_shared_from_this to obtain a shared_ptr from this.
